# POS Order 

## 概觀
練習以 ReactJS + Flux 實作 POS 點單介面


## 開發
1. 套件相依安裝 `npm install`
2. 運行服務 `npm start`
3. 開啟瀏覽器

```
http://localhost:8008
```

## 產出 js & html
運行 `npm run prod`

> 產出的 js & html file 會在根目錄下
